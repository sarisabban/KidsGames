# About:
This is a collection of HTML web browser games written using bare-metal HTML and JavaScript, no need to install anything, just open the file and the game should start on any web broswer

# How it works:
1. Choose a game and download it
2. Double click the file
The game should open a web browser and start working

# The games:

|File name  | Description                                           |
|-----------|-------------------------------------------------------|
|typing.html| A game the teaches how to type on a computer keyboard |
